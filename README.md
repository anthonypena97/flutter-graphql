## Installation

- Install xCode
- Install Flutter
- Download this repository
- Open Terminal in folder directory
- Run `flutter pub get`, in order to download packages

## Usage

- Open Terminal in folder directory
- Open an Adroid Simulator or iOS Simulator
- Run`flutter run`
